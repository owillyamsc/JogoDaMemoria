Memory Chess

Regras do jogo:

1. Objetivo: Encontre todos os pares de cartas iguais.
2. Preparação: Cartas são colocadas de cabeça para baixo em uma grade.
3. Jogada: Vire duas cartas por vez. Se forem iguais, ganhe um ponto. Caso contrário, vire-as de volta.
4. Memorização: Lembre-se das cartas que você virou para fazer as combinações.
5. Continuação: O jogo continua até todas as cartas serem combinadas.
6. Estratégia: Concentre-se e lembre-se das posições das cartas.
7. Variantes: Existem diferentes tamanhos de grade e regras adicionais.
8. Jogo Online: Encontre todos os pares o mais rápido possível.
